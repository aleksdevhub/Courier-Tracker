# Courier Tracker

Простое и удобное настольное приложение для курьеров, имитирующее мобильный интерфейс. Позволяет вести ежедневный учет рабочей активности, заказов и финансовой статистики.

## 📱 Описание проекта
Приложение разработано на платформе Windows Forms и имеет фиксированный вертикальный размер (380x640), повторяющий экран смартфона. Интерфейс реализован в виде пошагового опросника (Wizard) на базе скрытого контейнера вкладок. Пользователь отвечает на один вопрос на каждом экране, что делает ввод данных удобным и защищает от ошибок. Все элементы управления и дизайн созданы динамически через исходный код (без использования визуального дизайнера).

## 🛠 Стек технологий
* **Язык программирования:** C# (.NET Framework / .NET)
* **Технология интерфейса:** Windows Forms (динамическая сборка UI кодом)
* **Архитектура:** Событийно-ориентированная (Event-Driven)

## 📊 Функционал приложения
Приложение последовательно собирает и обрабатывает следующие данные:
* Профиль сотрудника (Имя, возраст, город)
* Логистические параметры (Тип транспорта, отработанные часы, пройденные километры)
* Результаты смены (Количество выполненных заказов)
* Финансовая отчетность (Чистый заработок и полученные чаевые)

На финальном экране приложение автоматически рассчитывает суммарный доход с учетом чаевых и распределяет метрики по визуальным карточкам.

## 📂 Структура приложения (Пошаговые экраны)
1. **Ввод личных данных** — Имя, возраст, город.
2. **Параметры смены** — Выбор транспорта (ComboBox), ввод заказов, часов и километров (NumericUpDown).
3. **Финансовый блок** — Ввод заработка и чаевых с безопасной валидацией (decimal.TryParse).
4. **Финальный отчет** — Экран результатов с крупным блоком итоговой суммы и плиточным разделением статистики.
